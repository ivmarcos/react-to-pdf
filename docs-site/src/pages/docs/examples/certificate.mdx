# Certificate Generator

Create professional certificates for courses, events, training programs, or achievements. This example demonstrates how to design an elegant certificate with custom styling.

## Complete Example

```tsx
import { usePDF } from 'react-to-pdf';

interface CertificateProps {
  recipientName: string;
  courseName: string;
  completionDate: string;
  instructorName: string;
}

function CertificateGenerator({
  recipientName,
  courseName,
  completionDate,
  instructorName
}: CertificateProps) {
  const { targetRef, toPDF } = usePDF({
    filename: `certificate-${recipientName.replace(/ /g, '-')}.pdf`,
    page: {
      format: 'a4',
      orientation: 'landscape',
      margin: 0
    },
    canvas: {
      scale: 3 // Higher quality for certificates
    }
  });

  return (
    <div className="certificate-container">
      <button
        onClick={() => toPDF()}
        className="download-btn"
      >
        Download Certificate
      </button>

      <div ref={targetRef} className="certificate">
        {/* Border */}
        <div className="certificate-border">
          {/* Header */}
          <div className="certificate-header">
            <h1>Certificate of Completion</h1>
          </div>

          {/* Body */}
          <div className="certificate-body">
            <p className="presented-to">This certificate is proudly presented to</p>

            <h2 className="recipient-name">{recipientName}</h2>

            <p className="completion-text">
              for successfully completing the course
            </p>

            <h3 className="course-name">{courseName}</h3>

            <p className="date-text">
              on <strong>{completionDate}</strong>
            </p>
          </div>

          {/* Footer */}
          <div className="certificate-footer">
            <div className="signature-section">
              <div className="signature-line"></div>
              <p className="signature-name">{instructorName}</p>
              <p className="signature-title">Course Instructor</p>
            </div>
          </div>

          {/* Decorative elements */}
          <div className="seal">
            <svg width="80" height="80" viewBox="0 0 80 80">
              <circle cx="40" cy="40" r="35" fill="#0ea5e9" opacity="0.1" />
              <circle cx="40" cy="40" r="30" fill="none" stroke="#0ea5e9" strokeWidth="2" />
              <text x="40" y="45" textAnchor="middle" fill="#0ea5e9" fontSize="14" fontWeight="bold">
                ✓
              </text>
            </svg>
          </div>
        </div>
      </div>
    </div>
  );
}

export default CertificateGenerator;
```

## Tips for Beautiful Certificates

1. **Use landscape orientation** - Certificates traditionally use landscape format
2. **High quality** - Set `scale: 3` or higher for print-quality output
3. **Serif fonts** - Georgia, Times New Roman work well for formal certificates
4. **Elegant spacing** - Give content room to breathe with generous padding
5. **Borders and seals** - Add visual authority with borders and verification seals
6. **Print-ready colors** - Use colors that print well (avoid very light colors)
